STEP 1 // INSTALL BOOTLOADER TO INTERNAL FLASH

HARDWARE INSTRUCTIONS

  1. Connect Daisy Seed via Micro USB to computer
  2. Press and HOLD the BOOT button on Daisy Seed
  3. While holding BOOT, press and RELEASE the RESET button
  4. Release the BOOT button
  5. LED should be solid (ROM bootloader mode)
  6. Click "Connect" button
  7. Browser window shows device: "DFU in FS mode paired"
  8. Click "Connect" in Browser window
  9. Click "Flash Bootloader"
Ready to connect
STEP 2 // UPLOAD FIRMWARE TO QSPI FLASH

HARDWARE INSTRUCTIONS - ENTER DAISY BOOTLOADER

  1. Power on HiChord
  2. Press the RESET button on Daisy Seed (short press)
  3. Within 1 second, press the BOOT button (short press)
  4. LED should pulse/blink (Daisy bootloader mode)
  5. Connect HiChord PCB via USB-C to computer
  6. Click "Connect" button
  7. Browser window shows device: "Daisy Bootloader" (You may need to unplug and re-connect USB-C for this to appear. Make sure HiChord PCB is powered on).
  8. Click "Connect" in Browser window
  9. Click "Program Firmware"
