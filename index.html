PRODUCTION BOOTLOADER + FIRMWARE UPLOADER

Firmware Build: 2025-12-21 11:12 (vREV 2.09) - Settings Reset Fix for Old PCBs
STEP 1 // INSTALL BOOTLOADER TO INTERNAL FLASH

HARDWARE INSTRUCTIONS

  1. Connect Daisy Seed via Micro USB to computer
  2. Press and HOLD the BOOT button on Daisy Seed
  3. While holding BOOT, press and RELEASE the RESET button
  4. Release the BOOT button
  5. LED should briefly blink fast then become solid red (ROM bootloader mode)
  6. Click "Connect" button
  7. Browser window shows device: "DFU in FS mode paired"
  8. Click "Connect" in Browser window
  9. Click "Flash Bootloader"
Ready to connect
STEP 2 // UPLOAD FIRMWARE TO QSPI FLASH

Enter Update Mode

  1. Power on HiChord
  2. Press RESET button on Daisy Seed
  3. Within 10ms, press BOOT button (timing critical)
  4. LED shows breathing/pulsing pattern
  5. Connect HiChord via USB-C to computer
  6. Click "Connect" button below
  7. Browser popup window appears
  8. Select "Daisy Bootloader" from device list
  9. Click "Connect" in popup window
  10. Click "Program Firmware" button below

⚠️ NOTE

If "Daisy Bootloader" does not appear: Unplug and reconnect USB-C. Ensure HiChord is powered on.
